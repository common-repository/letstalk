Add a social space to your website with topishare's Let's talk button.

Make your visitors life easier socializing on your website.

== Description ==

[topishare.com](https://topishare.com) lets you integrate your own customized social space on your website. It's easy and Free.

With your let's talk button installed, everyone can post, comment and message from your website. Easy to grow and engage communities. Use it for your fans, readers and organization.


To have a topishare Let's talk button on your wordpress site, follow these steps

1. Join topishare
2. Create your social space
3. In the 'Embed' tab, copy the code to the clipboard
4. Paste the code in the plugin settings page
5. Save settings

Now the Let's talk button should appear at the end of every post


== Installation ==

1. Upload `letstalk` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on Letstalk button settings
4. Paste the code you've copied form topishare site
5. Now you should see the let's talk button at the end of every post
